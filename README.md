# Blackjack Game 🎮

Juego de Blackjack desarrollado con JavaScript utilizando Vite.

## 🚀 Tecnologías utilizadas

- JavaScript
- HTML5
- CSS3
- Vite

## 📌 Funcionalidades

- Generación aleatoria de cartas
- Sistema de puntaje
- Turno del jugador
- Turno automático de la computadora
- Reinicio de partida

## ▶️ Cómo ejecutar el proyecto

1. Clonar el repositorio git clone " https://github.com/DanielFloreTrejo/blackjack-javascript.git"
2. Instalar dependencias "npm install"
3. Ejecutar el proyecto "npm run dev"
